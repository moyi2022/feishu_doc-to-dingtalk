📄 飞书→钉钉文档迁移工具

一键迁移飞书文档到钉钉文档系统

⚙️ API 配置

飞书凭证 前往飞书开放平台 →

💡 获取路径：飞书开放平台 → 我的应用 → 凭证与基础信息
💡 同上页面，点击「查看」获取
📋 需要开通的权限：
  • 查看新版文档
  • 查看知识库
  • 下载云文档中的图片和附件

钉钉凭证 前往钉钉开放平台 →

💡 获取路径：钉钉开放平台 → 应用开发 → 企业内部开发 → 凭证与基础信息
💡 同上页面，ClientSecret 需要点击查看
💡 获取路径：钉钉开放平台 → 首页 → 组织信息 → CorpId
💡 重要：这里需要填写 unionId，不是 userId！
🔧 点击展开：自动获取 unionId ▼
所需权限（在钉钉开放平台开通）：
  • 通讯录只读权限 Contact.User.Read
  • 通过手机号获取用户信息 qyapi_get_member_by_mobile
  • 成员信息读权限 qyapi_get_member
📋 需要开通的权限：
  • 知识库文档写权限
  • 企业存储文件写权限

🚀 开始迁移

💡 打开飞书文档 → 复制浏览器地址栏 URL
💡 打开钉钉知识库 → 查看浏览器URL：
https://alidocs.dingtalk.com/i/nodes/知识库ID/... 中间的部分
💡 打开钉钉知识库中的文件夹 → 查看URL末尾的 nodeId
💡 钉钉知识库 → 模板 → 复制模板ID（可选）
💡 打开飞书知识库 → 查看URL：
https://your-company.feishu.cn/wiki/知识库ID?... 中间的部分
💡 打开钉钉知识库 → 查看URL：
https://alidocs.dingtalk.com/i/nodes/知识库ID/... 中间的部分
💡 控制迁移子文档的层级深度
